Formatear

Se ha validado su solicitud para Formatear, se informa que se ha realizado las siguientes actividades para dar solución a su requerimiento:

1. Se identifica fallas de sistema operativo provocando que se congele de forma constante.
2. Se realiza una copia de la información en el equipo.
3. Se reinstala de nuevo el sistema operativo con la última versión de Windows 10.
4. Se configura el equipo con la configuración establecidas por el ICBF.
5. Se configura perfil de usuario y aplicativos correspondientes.
6. Se instala y se configura la impresora.

Una vez ejecutadas se procedió a realizar pruebas de funcionalidad evidenciado que queda operativo.
